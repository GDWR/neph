## Run Locally

Prerequisites:
- [Yarn installed](https://classic.yarnpkg.com/lang/en/docs/install/#debian-stable)
- [Rust and Cargo installed](https://doc.rust-lang.org/cargo/getting-started/installation.html)

Run development application
```shell
yarn install
yarn run dev tauri
```
